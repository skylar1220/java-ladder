## 기능명세서
- [x] 참여자 이름을 입력받는다.
  - [x] 쉼표 기준으로 구분한다.
  - [x] 5글자 이하인지 확인한다.
  - [x] 구분자가 연속해서 포함되지 않는다.
  - [x] 구분자로 시작하거나 끝나지 않는 지 확인한다.
  - [x] 공백을 입력하지 않는다.
- [x] 사다리 높이를 입력받는다.
  - [x] 1이상인지 확인한다.
  - [x] 정수인지 확인한다.
  - [x] Integer범위 이내의 숫자인지 확인한다.
- [x] 사다리를 생성한다.
  - [X] 한 층의 다리를 랜덤으로 생성한다.
  - [x] 사다리 높이만큼 생성한다.
  - [x] 사다리의 가로 라인이 겹치지 않도록 생성한다.
- [x] 실행 결과를 출력한다.
